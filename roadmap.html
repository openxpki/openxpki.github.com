Roadmap / Releases

Release Cycle

We do not have an official release cycle but in the past we managed to have one new minor release per quarter and we are expecting to keep this. Important security fixes or severe bugs will be published outside this timeline.

Versioning

We follow the semantic versioning guidelines, so you are able to update within the 3.x branch at any time without touching your config.

Packaging / OS Support

Community packages are provided for free for Debian Buster and available from our the package server. There is also a docker container available, see the Ressources section on the main page and the quickstart manual for details.
Native packages for other distributions are available via Enterprise-Support from White Rabbit Security.

Features / Roadmap

Version 3.x

This is the current stable and recommended release.

  • Upgrade to OpenSSL 1.1
  • Community Packages for Debian Buster
  • Finalise migration to LibSCEP (remove old SCEP tools)
  • Full support for EST
  • New WebUI

Version 2.x

Development has been discontinued, security fixes are still available via subscription.

  • Enhanced Reporting and Statistics
  • Enhanced command line tools (ACLs, parseable output)
  • New SCEP Layer (support for Engine and ECC)
  • New Command API allowing custom plugins/extensions
  • Transparant database layer for easy extension of tables

Version 1.x

Development has been discontinued

  • RPC API for request/renewal of certificates
  • Enrollment over Secure Transport (RFC 7030)
  • Dedicated renewal workflow
  • Command line API with granular ACL
  • Integrity protected audit log
  • Configuration Signing
