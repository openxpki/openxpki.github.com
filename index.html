OpenXPKI Core Features

Modern WebUI

The Ember.js based web frontend runs in all major browsers and provides an easy access to the system for users, operators and administrators.

Automation

Besides the standard enrollment protocols SCEP, EST, SimpleCMC and ACME a powerful REST-like API with OpenAPI support is also included.

Configuration

Full system configuration is held in YAML files. An overlay mechanism allows easy management of environment specific differences.

Flexible Crypto Layer

Crypto operations are based on the renowned OpenSSL toolkit and can utilize almost any compatible Hardware Security Modules (HSM).

Multiple Backends

Operate your CA signing keys on a remote system or even delegate certificate issuance to an external CA like Digicert, Sectigo or SwissSwign.

SubCAs and Rollover

Run multiple separate CAs within a single installation and enjoy a fully automated rollover of CA generations as operational standard task.

Integration

A generic API allows an easy integration of existing CMDB and ITSM systems to automate request validation, approvals and notifications.

User Management

Seamlessly integrate your existing Identitiy and Access Management using SAML, OAuth, LDAP or webserver based SSO solutions.

Workflow Driven

Processes are driven by workflows with the definitons beeing part of the customer configuration which makes adoption to specific project needs easy.

Easy Deployment

Install the software via your distributions package manager, copy and adjust the sample configuration, load your key material and you are ready to go!

Free Open Source

The fully functional software with an extensive example configuration is provided under the Apache License with a friendly support via mailing list.

Enterprise Ready

Consulting, Configuration, Packaging and operational support with SLA is available directly from the core developers via White Rabbit Security GmbH.
